# 📧 Spam Email Classifier

A Machine Learning project that classifies SMS/email messages as **Spam** or **Ham** using Natural Language Processing (NLP).

---

## Features

- Text preprocessing using NLTK
- TF-IDF Vectorization
- Multinomial Naive Bayes Classifier
- Streamlit Web Application
- Interactive prediction interface

---

## Technologies Used

- Python
- Pandas
- Scikit-learn
- NLTK
- Streamlit
